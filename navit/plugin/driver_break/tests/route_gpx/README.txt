Hiking test routes (GPX)
========================

These GPX files define test routes for the Driver Break plugin. Waypoints are
DNT huts/cabins with name and elevation so they appear in height profile data.
Use with energy-based routing in Navit (cycling/hiking profile with use_energy_routing).

Main route: Mogen -> Tråastølen -> Finsehytta -> Myrdal Fjellstove
  OSM: start node 1356402304, via 872147024, via 444092409, end way 1012805346

Alternative: Gjendesheim -> Vetledalseter
  OSM: start node 791106132, end node 1356403887

Generated by generate_hiking_route_gpx.sh (run from tests/ or build dir).
